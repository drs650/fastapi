# KTX 역 데이터 전처리 및 데이터 파이프라인 구축 프로젝트


## 내용 요약

### 1. 원본 데이터 수집 및 인코딩 처리
- `os.getcwd()`를 사용하여 경로(`BASE_DIR`)를 설정했습니다.

### 2. 데이터 전처리 및 결측치 검증
- 데이터 로드 후 `.isnull().sum()` 을 이용하여 결측치 유무를 확인했습니다.

### 3. 주소 기반 파생 컬럼 생성
- 길게쓰여진 주소를 구분짓기위해 (서울, 경기, 충청, 강원, 경상, 전라) 6개의 지역으로 나눠서 분류하였습니다.
- (`do_list`)리스트를 만들고 특정 단어를 따와서 새로운 파생 컬럼을 추가했습니다.

### 4. PostgreSQL 데이터베이스 저장
- 총 102개 행의 데이터를 `ktx_raw` 테이블에 저장 완료했습니다.

### 5. 정제 데이터 CSV 백업 저장
- 최종 데이터프레임을 `output/ktx_lot.csv` 경로에 한글 깨짐이 없는 `utf-8-sig` 인코딩 형식으로 저장했습니다.
